# Workflow: Add UI Section

1) Create `src/ui/sections/<SectionName>/`
2) Add required files:
   - `index.ts`
   - `section.ts`
   - `state.ts`
3) (Optional) Add `styles.css.ts` and/or `parts/`
4) Register the section in the sections registry

Note: follow `.claude/rules/ui.sections.md` for file responsibilities, state rules, and lifecycle contract.
